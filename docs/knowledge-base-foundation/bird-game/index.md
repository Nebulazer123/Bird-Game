## Top Direction Decisions
- Zen Soarer is now the default product direction.
- Challenge mode is preserved as a secondary mode for mastery play and regression safety.
- Zen's main loop is note discovery plus nest composition, not ring clearing.
- Projectile combat is no longer the default product identity; Zen uses lighter pressure and Wind Pulse instead.
- The current modular repo structure remains the foundation. The next work is polish, not a restart.

## Top Tooling Decisions
- Controller support: Web Gamepad API + small input normalizer (PlayStation-first, Xbox supported); DS4Windows optional fallback on Windows.
- Feel iteration: add Tweakpane for sliders/toggles on key tuning values.
- Audio: add Howler.js for quick, clear SFX + wind ambience.
- Safari coverage: treat Safari as best-effort; use Chrome as baseline and optionally add Playwright WebKit smoke tests.
