# Project Overview

## Summary
Bird Game is an existing 3D browser game prototype currently presented as Featherwind Valley. It already has a playable third-person bird flight slice with rings, enemy hunter birds, stage progression, upgrades, art assets, debug tools, and automated Playwright coverage. The project is now entering a serious planning pass to decide what the game should become next rather than assuming the current loop is the right long-term answer.

## Project Type
Existing browser game repo and vertical-slice redesign/planning project.

## Target User
Broad action-game players who want an immediately exciting, readable, expressive bird-flight experience without needing flight-sim patience or deep technical mastery.

## Problem Being Solved
The current build proves the team can make a playable bird game, but it does not yet answer the higher-value product question: what bird fantasy is strongest, what loop best serves that fantasy, and which existing systems deserve to stay versus be replaced.

## Scope Snapshot
This planning pass focuses on product direction, loop quality, fantasy clarity, structural viability, readability, and specialist handoff. It does not lock the team into preserving the current combat, stage flow, progression stack, or monolithic code structure.
